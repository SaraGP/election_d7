Election
========
Election is a module for Drupal 7.x providing online voting in STV elections and referendums.

Dependencies
------------
* Entity API (http://drupal.org/project/entity)
* Views (http://drupal.org/project/views)
* Date (http://drupal.org/project/date)

Installation
------------
Enable at least these three modules and their dependencies:
   election
   election_post
   election_candidate
